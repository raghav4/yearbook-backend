# Backend of Yearbook Project

Backend Application of [Yearbook](https://github.com/raghav4/yearbook)

## Please ensure you have the following installed:

- [node.js]("https://nodejs.org/en/") `v12.14.1`
- [npm]("https://www.npmjs.com/") `v6.14.2`
- [mongodb]("https://www.mongodb.com/")

## Installation Guide

- Install all the dependencies by running `npm install`.
- Update the environment variables by creating a `.env` file similar to `.env.example`
- Run command `npm start` to start the server.
- Visit the server at [http://localhost:3000](http://localhost:3000)
